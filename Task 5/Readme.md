## 🌟 Task 5 – Sales Data Analysis (Pandas)
A simple Sales Data Analysis Project built using Python, Pandas, and Matplotlib for the Elevate Labs Internship.
<br>
This task demonstrates:
- Loading CSV data using Pandas
- Performing groupby(), sum(), and filtering
- Creating charts using Matplotlib
- Understanding the basic data analysis workflow

### 🚀 Features
- Load sales data from a CSV file
- Display the first few rows of the dataset
- Calculate total sales per product
- Analyze monthly sales trends
- Create bar charts and line charts
- Clean and analyze real-world structured data
