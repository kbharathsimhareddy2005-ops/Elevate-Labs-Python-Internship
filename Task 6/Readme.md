## 🌐 Task 6 – Portfolio Website (Flask)
A Personal Portfolio Web Application Built with Flask
<br>
This project is part of the Elevate Labs Internship – Task 6, where the goal is to create a Flask-based portfolio website showcasing your skills, projects, certificates, and resume.
<br>
This version includes:
- Flask routing
- Home, About, Skills, Projects & Contact sections
- Clean and responsive design
- Organized folder structure (templates + static files)

## 🌟 Features
### Modern Multi-Section Portfolio

Includes the following sections:
- Home (with profile photo)
- About Me
- Skills
- Projects
- Certificates
- Contact Form

## ▶️ How to Run the Project
### 1️⃣ Install dependencies
```bash
pip install flask
```
### 2️⃣ Run the Flask app
```bash
python app.py
```
### 3️⃣ Open in Browser
```bash
http://127.0.0.1:5000/
```

## 🧩 Flask Routes Overview
```bash
| Route      | Method | Purpose                             |
| ---------- | ------ | ----------------------------------- |
| `/`        | GET    | Load portfolio homepage             |
| `/contact` | POST   | Process contact form & save message |
```
